# AyaNasfi-appdev-reachingout.md

## Description of the problem

The **reachingout** challenge focuses on network communication from an Android application.  
The goal is to interact with a local HTTP server, send the expected requests, and retrieve the flag from the server response.

## Solution

I solved this challenge by creating an Android application that communicates with the server using HTTP GET and POST requests.  
When the app starts, it connects to the server, follows the given endpoint, and sends the required parameters.
Once the correct values are sent, the server returns the flag.  
The result is logged using the 'MOBISEC' tag and also displayed on the screen.
